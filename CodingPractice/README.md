# 코딩 연습 - 인덱서

교안의 예제 코드를 직접 따라 작성하고 실행하는 프로젝트입니다.

> 각 과제의 코드를 `Program.cs`에 작성하세요.
> 이전 과제의 코드가 남아있으면 변수 이름이 겹쳐서 오류가 발생할 수 있습니다. 이전 과제의 코드를 지우거나 주석 처리한 후 다음 과제를 진행하세요.

---

## 과제 1: 기본 인덱서 예제

정수 인덱스에 따라 짝수/홀수 문자열을 반환하는 `Catalog` 클래스를 만들고, 인덱서를 통해 값을 출력하세요.

- `Catalog` 클래스에 정수형 인덱서(`this[int index]`)를 정의합니다
- 인덱스가 짝수이면 `"{index}: 짝수 반환"`, 홀수이면 `"{index}: 홀수 반환"`을 반환합니다
- 인덱스 0, 1, 2로 호출하여 출력합니다

**실행 결과**

```
0: 짝수 반환
1: 홀수 반환
2: 짝수 반환
```

## 과제 2: 배열 필드를 감싸는 인덱서

내부에 배열 필드를 가지는 `Car` 클래스를 만들고, 인덱서를 통해 외부에서 배열처럼 접근하세요.

- `Car` 클래스에 `string[]` 필드, 생성자(배열 크기), `Length` 속성, 인덱서를 정의합니다
- 크기 3으로 생성한 후 인덱서로 "CLA", "CLS", "AMG"를 저장합니다
- `for` 문으로 순회하며 출력합니다

**실행 결과**

```
CLA
CLS
AMG
```

---

## 과제 3: 요일 클래스 예제

인덱서와 생성자를 함께 활용하는 `Week` 클래스를 구현하세요.

- `Week` 클래스에 기본 생성자(크기 7)와 매개변수 생성자를 정의합니다
- 읽기 전용 `Length` 속성과 인덱서를 정의합니다
- 크기 3으로 생성한 후 "일요일", "월요일", "화요일"을 저장하고 출력합니다

**실행 결과**

```
일요일
월요일
화요일
```

## 과제 4: 문자열 인덱서

문자열을 인덱스로 사용하는 `NickName` 클래스를 구현하세요.

- `using System.Collections;`를 추가합니다
- `NickName` 클래스에 `Hashtable` 필드와 문자열 인덱서(`this[string key]`)를 정의합니다
- "홍길동" → "RedPlus", "김철수" → "BlueStar"를 저장하고 출력합니다

**실행 결과**

```
RedPlus, BlueStar
```

---

## 과제 5: 다중 인덱서

정수 인덱서와 문자열 인덱서를 모두 가지는 `MultiIndexer` 클래스를 구현하세요.

- `using System.Collections;`를 추가합니다
- `MultiIndexer` 클래스에 `Hashtable` 필드, 정수 인덱서, 문자열 인덱서를 정의합니다
- 정수 인덱서로 값을 저장하고, 문자열 인덱서로도 값을 저장한 후 모두 출력합니다

**실행 결과**

```
multi[0] = 첫 번째 값
multi["name"] = 홍길동
multi["city"] = 서울
```

## 과제 6: 인덱서와 속성의 비교

속성과 인덱서를 함께 가지는 `Example` 클래스를 구현하세요.

- `Example` 클래스에 
- `Name` 속성(단일 값)과 인덱서(배열 형태 접근)를 정의합니다
- 속성으로 이름을 설정하고 출력한 후, 
- 인덱서로 값을 저장하고 출력합니다

**실행 결과**

```
홍길동
첫 번째, 두 번째
```
